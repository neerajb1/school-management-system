# School Management System - Frontend

This is the frontend part of the School Management System project built using React. 

## Project Structure

- **public/**: Contains the static files for the React application.
  - **index.html**: The main HTML file that serves as the entry point for the application.

- **src/**: Contains the source code for the React application.
  - **components/**: Contains reusable React components.
    - **App.js**: The main component that sets up routing and renders other components.
  - **pages/**: Contains the different pages of the application.
    - **HomePage.js**: A sample page for the application.
  - **services/**: Contains functions for making API calls to the backend.
    - **api.js**: Functions for interacting with the backend API.
  - **index.js**: The entry point for the React application that renders the App component.

## Getting Started

1. **Install Dependencies**: 
   Navigate to the `frontend` directory and run:
   ```
   npm install
   ```

2. **Run the Application**: 
   After installing the dependencies, you can start the application with:
   ```
   npm start
   ```

3. **Access the Application**: 
   Open your browser and go to `http://localhost:3000` to view the application.

## API Integration

The frontend communicates with the backend API to fetch and manage data. Ensure that the backend server is running before testing the frontend.
